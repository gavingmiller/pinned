Pinned is a [Pinboard API (v1.0)](http://pinboard.in/api/) wrapper. 

Pinned is currently in active development, and any methods, classes, or
structure are liable to changes.

## Installation

    gem install pinned

## Usage

### Get all posts

    require 'pinned'

    pinned = Pinned.create(username, password)
    pinned.posts_all
